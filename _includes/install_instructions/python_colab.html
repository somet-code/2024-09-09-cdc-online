Python

Python is a popular language for research computing, and great fo rgeneral-purpose programming as well. We will use Google Colaboratory to write and execute Python code using notebooks.

You will need to have a Google account and a device that can access Google Colaboratory and Google Drive in a reasonably up-to-date major browser of your choice. Google Colab has been tested with the latest versons of Google Chrome, Mozilla Firefox, and Apple Safari browsers and therefore these are recommended for use.
